# Git Commands Cheat Sheet for Speak Easy Project

## 🚀 **Getting Started**
Your Git repository is now set up and ready to use!

## 📝 **Daily Commands**

### **Check Status**
```bash
git status                    # See what files have changed
git diff                      # See detailed changes in files
```

### **Save Your Work**
```bash
git add .                     # Stage all changes
git add filename.htm          # Stage specific file
git commit -m "Description"   # Save changes with message
```

### **View History**
```bash
git log --oneline            # See commit history (short)
git log                      # See detailed commit history
git log --graph              # See commit history with branches
```

### **Update from Remote (if you add GitHub later)**
```bash
git pull                     # Get latest changes
git fetch                    # Check for updates
```

## 🔄 **Workflow Example**

1. **Make changes** to your files
2. **Check what changed**: `git status`
3. **Stage changes**: `git add .`
4. **Save changes**: `git commit -m "What I changed"`
5. **Repeat!**
