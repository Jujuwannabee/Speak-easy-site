# Speak Easy - Professional English Learning Platform

## Overview
Speak Easy is a professional English language learning platform designed for primary, middle, and high school students. The platform offers interactive video courses with native English speakers to improve vocabulary and oral expression skills.

## Features
- **Professional Design**: Modern, business-ready interface with responsive design
- **Interactive Learning**: Three-step learning process with video preparation, vocabulary building, and live practice
- **Native Speakers**: Direct interaction with bilingual instructors
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations and hover effects

## Technology Stack
- **HTML5**: Semantic markup with proper accessibility
- **CSS3**: Modern styling with CSS Grid, Flexbox, and responsive design
- **JavaScript**: Interactive functionality and form validation
- **Responsive Design**: Mobile-first approach with breakpoints

## Pages
1. **Home Page** (`Home page.htm`) - Main landing page with hero section and learning objectives
2. **About** (`About.htm`) - Company mission and team information
3. **Program Description** (`Program_description.htm`) - Detailed learning methodology
4. **Login** (`Log_in.htm`) - User authentication interface

## Getting Started
1. Open `Home page.htm` in your web browser
2. Navigate through the different sections using the header menu
3. All pages are interconnected and maintain consistent navigation

## Customization
The website uses a centralized CSS file (`style.css`) for easy customization:
- Colors can be modified in the CSS variables
- Layout adjustments can be made in the grid and flexbox sections
- Typography can be updated in the font-family declarations
